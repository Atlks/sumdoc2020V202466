Overview概论	
主要为了解决目前需要集成一些成品软件或插件作为内部模块的时候，没有合适的接口的问题。。一般软件最起码还是有gui接口的。这个解决方案就是为了通过gui接口来集成调用其他模块软件的问题。。。
此外通过pc手机模拟器，还可以集成大量的手机应用app。。

应用场景：
比如集成qq作为消息发布器模块。。Qq本身是没有提供api与cli接口的，只能通过gui接口集成。
比如自动登录或操作某些有安全控件的网站或者程序

一般很多程序可以通过pc模拟器来加载手机应用操作。但是一些app必须要在手机上才可操作。
